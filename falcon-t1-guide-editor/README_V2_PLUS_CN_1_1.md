# Falcon T1 Guide Editor - V2 Plus 中文版 1.1

本版本继续基于 V2 Plus 中文版迭代，不改变原有主要操作逻辑。

## 本次新增 / 修复

1. 从下一页拉内容到当前页时，默认放在当前页最后一个视觉元素后面，不再随机插入。
2. 新增对齐基准线：打开“鼠标拖拽/缩放”后，移动元素时会显示页面中心线、左右安全线、元素中心线。
3. 新增批量拓展元素功能：
   - 批量放大当前页元素
   - 批量缩小当前页元素
   - 批量拉宽当前页元素
   - 统一当前页间距

## 推荐使用

如果需要重新分配社媒页面内容：

1. 先选中当前页。
2. 点击“从下一页拉内容到当前页”。
3. 新拉上来的内容会自动放在当前页最后一个内容块后面。
4. 如果页面仍然空，可以点击“填充当前页空白”或“批量放大当前页元素”。
5. 拖拽微调时，打开“鼠标拖拽/缩放：开”和“对齐基准线：开”。

## 注意

- 批量功能作用于当前页，不会处理页脚。
- 对齐基准线只有在鼠标拖拽/缩放开启后才会显示。
- 如果你还有第 4 个需求，可以继续在这个版本上迭代。
