# Falcon T1 Guide Editor - V2 Plus 中文版

这是基于 V2 稳定版继续迭代的版本，尽量保留 V2 原有功能，同时新增社媒排版功能。

## 打开方式

1. 用 VS Code 打开整个 `Falcon_T1_Guide_Complete` 文件夹。
2. 右键 `editor.html`。
3. 选择 `Open with Live Server`。
4. 点击顶部 `加载 index.html`。

## 保留的 V2 功能

- 新增卡片
- 新增图片卡片
- 选中外层卡片
- 选中表格
- 删除选中元素
- 图片列表识别
- 表格列表识别
- 修改文字内容
- 修改字体、字号、颜色、字重、对齐、行高
- 修改卡片背景、边框、圆角、宽高、位置
- 替换图片
- 插入图片
- 图片铺满父级卡片
- 修改表格大小和样式
- 鼠标拖拽 / 缩放
- 下载 HTML

## 新增功能

- 新增文字
- 新增页面
- 应用 1080×1350 社媒尺寸
- 更新页码
- 清理当前页位置
- 填充当前页空白
- 将选中内容移到上一页
- 将选中内容移到下一页
- 从下一页拉内容到当前页
- 页面列表
- 导出当前页 PNG
- 导出全部 PNG

## 使用建议

如果你要做 Facebook / Instagram 社媒轮播图：

1. 先点 `应用 1080×1350 尺寸`
2. 点 `更新页码`
3. 逐页检查空白
4. 空白太多时用 `填充当前页空白` 或 `从下一页拉内容到当前页`
5. 需要补充说明时用 `新增文字`
6. 完成后点 `导出当前页 PNG` 或 `导出全部 PNG`

## 注意

- PNG 导出依赖 html2canvas CDN，需要网络能访问 `cdn.jsdelivr.net`。
- 如果导出失败，请先点击 `下载 HTML` 保存当前修改。
- 本地压缩包里的 `editor-v2-original.html` 是 V2 原版备份。
