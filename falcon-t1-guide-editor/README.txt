Falcon T1 UV Crystal Guide project

Open index.html in a browser to preview.
Upload this folder to StackBlitz as a Static HTML project.
All images are stored in the images folder and referenced by relative paths.

Visual editor added:
1. Open editor.html through StackBlitz / VS Code Live Server / any static web server.
2. Click Load index.html.
3. Click an image to resize or replace it by dragging a new image into the replacement area.
4. Edit text directly in the preview.
5. Click Select parent .card / .page to edit the outer card / border box.
6. New version supports adding new cards.
7. New version supports inserting an image into a card and making the image fill the card.
8. Click Download HTML and rename the downloaded file to index.html when you want to replace the original page.

Tips:
- If you want a full-bleed image card, create a new image card or select an image and click "让图片铺满父级卡片".
- If a card needs image fill mode to work well, give the card a fixed height first.

Note: Some browsers block fetch('index.html') when opening editor.html directly from local disk. If this happens, use VS Code Live Server, StackBlitz, or manually import index.html from the left panel.
