# Social Carousel Editor V3

Open `social-editor-v3.html` with VS Code Live Server.

## Main features

- Load `social-carousel.html`
- Add text blocks
- Add cards and image cards
- Delete / duplicate selected elements
- Select slide, card, or table
- Fill blank area
- Move selected content to previous / next slide
- Pull content from the next slide
- Renumber slides
- Export current slide as 1080 × 1350 PNG
- Export all slides as PNG
- Download the edited HTML

## Important

`social-carousel.html` may be very large if it contains inline base64 images. If upload through ChatGPT fails, upload that file manually through GitHub:

`falcon-t1-guide-editor/social-carousel.html`

The editor can also import a local `social-carousel.html` from the left import area.
