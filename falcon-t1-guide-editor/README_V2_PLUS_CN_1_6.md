# Falcon T1 Guide Editor - V2 Plus 中文版 1.6

本版本基于 1.5 继续迭代。

## 本次新增

### 修复当前图片显示

新增按钮：`修复当前图片显示`

适合处理这些情况：

- 某张图片导出后变形
- 某张图片被裁切
- 某张图片被拉伸
- 某张图片显示位置不对
- 图片被“铺满卡片”后想恢复完整显示

## 使用方法

1. 选中异常图片。
2. 如果点不中图片，可以左侧点击 `重新扫描图片`，再从图片列表里选择。
3. 点击右侧图片操作区的 `修复当前图片显示`。
4. 再导出当前页 PNG。

## 这个按钮会做什么

- 图片完整显示
- 图片居中
- 图片不拉伸
- 取消固定高度
- 尽量解除外层卡片裁切
- 把 `object-fit` 改为 `contain`
- 把 `object-position` 改为 `center center`

## 保留 1.5 功能

- PNG 自动下载失败时，提供手动下载链接
- 克隆干净 1080 × 1350 页面导出
- 大预览模式
- 对齐基准线
- 批量放大 / 缩小 / 拉宽 / 统一间距
