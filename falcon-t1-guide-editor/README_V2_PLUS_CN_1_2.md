# Falcon T1 Guide Editor - V2 Plus 中文版 1.2

本版本继续基于 V2 Plus 中文版 1.1 迭代。

## 本次新增 / 修复

1. 修复导出单张 PNG 时图片内容可能不显示的问题。
   - 导出前会等待图片加载。
   - 会临时把图片转成内嵌 data URL 再导出，降低图片空白概率。
   - 导出后会恢复原始图片路径。

2. 放大中间预览界面。
   - 默认缩放从 80% 改为 100%。
   - 缩放范围改为 60% 到 180%。
   - 中间画布边距缩小，编辑空间更大。

## 使用建议

如果要导出社媒图：

1. 确认页面中的图片在预览里已经显示。
2. 点击“导出当前页 PNG”。
3. 左侧状态栏会显示“正在准备图片资源”。
4. 等浏览器自动下载 PNG。

如果导出的图片仍然为空：

1. 先点击“下载 HTML”保存当前版本。
2. 把下载文件改名为 `index.html`。
3. 放回项目文件夹覆盖原文件。
4. 重新用 Live Server 打开 editor.html 后再导出。
